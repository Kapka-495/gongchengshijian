# 精准化学数据平台 (PIChemData) — 后端接口文档

> 前端基地址：`/pichemdata/api/v1`（代理到 `https://aisci.ustc.edu.cn`）
> 所有接口统一响应格式：`{ code: 200, msg: null, data: ... }`
> 分页接口 data 格式：`{ result: [...], page: { size, current, total } }`

---

## 一、认证模块 (Auth)

### 1.1 登录
```
POST /auth/login
Content-Type: application/json

Request:
{
  "username": "admin",
  "password": "admin123"
}

Response data:
{
  "token": "jwt-token-string",
  "userInfo": {
    "id": 1,
    "username": "admin",
    "email": "admin@ustc.edu.cn",
    "role": "admin",
    "nickname": "管理员",
    "phone": "13800138000",
    "avatar": ""
  }
}
```

### 1.2 注册
```
POST /auth/register
Content-Type: application/json

Request:
{
  "username": "newuser",
  "password": "123456",
  "email": "newuser@ustc.edu.cn",
  "nickname": "新用户"
}

Response data:
{
  "success": true
}
```

### 1.3 获取当前用户信息
```
GET /auth/userinfo
Authorization: Bearer <token>

Response data:
{
  "id": 1,
  "username": "admin",

```

```

```

### 1.5 修改密码
```
POST /auth/change-password
Authorization: Bearer <token>

  "oldPassword": "admin123",

  "success": true

```


## 五、分子模块 (Molecule) — 开放接口 (Open API)

### 5.1 分子核心检索（主搜索接口）
```
POST /molecule/open/search/core
Content-Type: application/json

Request:
{
  "page": 1,
  "size": 10,
  "keyword": "",        // 关键词（匹配 formula / smiles / inchi / tags）
  "type": "",           // 分子类型：无机小分子 | 有机小分子 | 芳香族化合物 | 无机盐 | 烯烃 | 无机酸
  "category": "",       // 分类筛选
  "formula": "",        // 精确分子式，如 "H2O"
  "smiles": "",         // SMILES 表达式
  "inchi": "",          // InChI 字符串（模糊匹配）
  "massMin": null,      // 最小分子量
  "massMax": null,      // 最大分子量
  "charge": null,       // 电荷
  "spin": null,         // 自旋
  "tags": [],           // 标签数组，如 ["溶剂", "水"]
  "sortField": "",      // 排序字段：id | mass | formula
  "sortOrder": "asc"    // asc | desc
}

Response data:
{
  "result": [
    {
      "id": 1,
      "formula": "H2O",
      "smiles": "O",
      "inchi": "InChI=1S/H2O/h1H2",
      "mass": 18.015,
      "volume": 18.07,
      "type": "无机小分子",
      "tags": ["水", "溶剂", "无机"],
      "charge": 0,
      "spin": 1,
      "createTime": "2025-12-01"
    }
  ],
  "page": { "size": 10, "current": 1, "total": 12 }
}
```

## 十三、XRD 衍射数据处理（独立 Python 后端）

> XRD 后端为 Python FastAPI，独立部署在 `http://localhost:8000`
> 前端通过 `/xrd-api` 代理访问（重写路径：`/xrd-api` → `/api`）
> **响应格式为 `{ original, processed }`，不经过 `{ code, msg, data }` 封装**

### 13.1 JSON 数据提交处理
```
POST /api/process          (前端请求 /xrd-api/process)
Content-Type: application/json

Request:
{
  "data": [
    { "angle": 25.3, "intensity": 100.0 },
    { "angle": 28.4, "intensity": 85.2 }
  ],
  "min_angle": 5.0,
  "max_angle": 90.0,
  "step": 0.01,
  "sigma": 0.1
}

Response:
{
  "original": {
    "angles": [25.3, 28.4],
    "intensities": [100.0, 85.2]
  },
  "processed": {
    "angles": [5.0, 5.01, 5.02, ...],
    "intensities": [0.001, 0.002, ...]
  }
}
```

### 13.2 NPY 文件上传处理
```
POST /api/process-npy      (前端请求 /xrd-api/process-npy)
Content-Type: multipart/form-data

Form fields:
  - file: .npy 文件（shape [N, 2]，列：角度、强度）
    或 0 维对象（含 "features" 键，shape [n_samples, n_grid]）
  - min_angle: 5.0
  - max_angle: 90.0
  - step: 0.01
  - sigma: 0.1

Response: 同 13.1
```

## 附录

### 统一错误响应格式
```json
{
  "code": 400,          // 或 401 / 403 / 404 / 500
  "msg": "错误描述信息",
  "data": null
}
```

### 认证方式
除登录、注册和公开 GET 接口外，其余接口需携带 JWT Token：
```
Authorization: Bearer <token>
```

### 分页参数约定
| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| page | int | 1 | 当前页码（从 1 开始） |
| size | int | 10 | 每页条数 |
| keyword | string | "" | 搜索关键词 |
