# 前端项目 API 接口清单

本文档根据 `frontend/src/api/index.js`、`frontend/src/utils/request.js`、`frontend/vue.config.js` 中的调用整理，供后端实现接口时参考。

## 全局约定

- 前端请求基址：`/pichemdata/api`
- 开发环境代理：`/pichemdata/api` 会被代理到 `http://localhost:8000/api`
- Axios 超时时间：`15000ms`
- 登录后请求头：`Authorization: Bearer <token>`
- 通用响应格式：

```json
{
  "code": 200,
  "msg": null,
  "data": {}
}
```

- 前端拦截器约定：`code` 必须为 `200` 才视为成功；`401`、`403` 视为认证失败。
- 文件上传接口使用 `multipart/form-data`。
- 多数接口当前带有 mock 降级逻辑，但 `USE_MOCK = false`，会优先请求真实后端。

## XRD 工具接口

XRD 接口使用 `fetch` 直接请求 `/pichemdata/api` 下的路径，开发环境同样会被代理到 `http://localhost:8000/api`。

| 方法 | 前端路径      | 后端实际路径      | 参数                                                        | 说明                            | 期望 data                                                    |
| ---- | ------------- | ----------------- | ----------------------------------------------------------- | ------------------------------- | ------------------------------------------------------------ |
| POST | `/process`    | `/api/process`    | JSON: `{ data, min_angle, max_angle, step, sigma }`         | 对文本 XRD 数据做处理           | `{ original: { angles, intensities }, processed: { angles, intensities } }` |
| POST | `/upload-npy` | `/api/upload-npy` | FormData: `file`, `min_angle`, `max_angle`, `step`, `sigma` | 上传 `.npy` 文件并处理 XRD 数据 | `{ original: { angles, intensities }, processed: { angles, intensities } }` |

### `/process` 请求体示例

```json
{
  "data": [
    [10.1, 100],
    [20.2, 85]
  ],
  "min_angle": 5,
  "max_angle": 90,
  "step": 0.01,
  "sigma": 0.1
}
```

## 后端实现优先级建议

1. 认证：`/api/login`、`/api/user-info`、`/api/register`、`/api/logout`
2. 首页与列表：新闻、公告、材料、文献、分子检索、统计接口
3. 分子详情与谱图：`/api/molecule/open/search/core`、`/api/molecule/open/spectrum/{picId}`
4. 文件上传：`/api/molecule/open/upload`、`/api/upload`
5. 管理后台：审计日志、元数据增删查
6. XRD：`/api/process`、`/api/upload-npy`
