采用python的sqlite3库进行数据库的维护。



### `molecules` 表

| 字段名 | 数据类型 | 含义 / 业务说明 |
| :--- | :--- | :--- |
| **主键与核心标识符 (Identifiers)** | | |
| `id` | INTEGER | 自增主键 |
| `inchikey` / `canonicalInchikey` | TEXT | InChIKey (唯一哈希字符串)，设了 `UNIQUE` 用于去重 / 标准化 InChIKey |
| `inchi` / `canonicalInchi` | TEXT | 国际化合物标识 (InChI) / 标准化 InChI |
| `smiles` / `canonicalSmiles` | TEXT | 简化分子线性输入规范 (SMILES) / 标准化 SMILES |
| `iupac` | TEXT | IUPAC 国际标准命名 |
| `cas` | TEXT | CAS 登记号 (Chemical Abstracts Service) |
| `cid` | INTEGER | 化合物 ID (通常对应 PubChem CID) |
| `title` / `molecularName` | TEXT | 分子通用名称 / 分子名 |
| `formula` / `displayFormula` | TEXT | 化学式 / 用于展示的化学式 |
| **基础理化性质 (Physicochemical Properties)** | | |
| `weight` | REAL | 分子量 (Molecular Weight) |
| `exactMass` / `monoisotopicMass` | REAL | 精确质量 / 单同位素质量 |
| `charge` | INTEGER | 形式电荷 |
| `xlogp` | REAL | 脂水分配系数 (计算值)，衡量疏水性 |
| `tpsa` | REAL | 拓扑极性表面积 (Topological Polar Surface Area) |
| `hbondDonorCount` / `hbondAcceptorCount`| INTEGER | 氢键供体数量 / 氢键受体数量 |
| `complexity` | REAL | 分子拓扑复杂度 |
| **拓扑与2D结构特征 (Topology & 2D Features)** | | |
| `heavyAtomCount` | INTEGER | 重原子数量 (非氢原子总数) |
| `isotopeAtomCount` | INTEGER | 同位素原子数量 |
| `rotatableBondCount` | INTEGER | 可旋转键数量 |
| `covalentUnitCount` | INTEGER | 共价单元数量 |
| `atomstereoCount` / `definedAtomStereoCount` / `undefinedAtomStereoCount` | INTEGER | 原子立体中心总数 / 明确数 / 未明确数 |
| `bondStereoCount` / `definedBondStereoCount` / `undefinedBondStereoCount` | INTEGER | 化学键立体中心(顺反异构)总数 / 明确数 / 未明确数 |
| **三维与构象特征 (3D & Conformer Features)** | | |
| `volume3d` | REAL | 3D 分子体积 |
| `conformerCount3d` | INTEGER | 3D 构象数量 |
| `conformermodelrmsd3d` | REAL | 构象模型的均方根误差 (RMSD) |
| `effectiverotorCount3d` | INTEGER | 3D 有效旋转键数量 |
| `featureringCount3d` | INTEGER | 3D 特征环数量 |
| `featureCount3d` | INTEGER | 3D 药效团特征总数 |
| `featureanionCount3d` / `featurecationCount3d` | INTEGER | 3D 阴离子特征数量 / 3D 阳离子特征数量 |
| `featuredonorCount3d` / `featureacceptorCount3d`| INTEGER | 3D 氢键供体特征数 / 3D 氢键受体特征数 |
| `featurehydrophobeCount3d` | INTEGER | 3D 疏水特征数量 |
| `xstericquadrupole3d` / `ystericquadrupole3d` / `zstericquadrupole3d` | REAL | 3D 空间四极矩的 X / Y / Z 分量 |
| **系统及业务元数据 (System & Metadata)** | | |
| `categoryObject` / `categoryLabel` | TEXT | 数据分类对象 / 分类标签 |
| `uniqueIdentification` | TEXT | 业务生成的 UUID 或系统级唯一标识符 |
| `publicIdentification` | INTEGER | 公开标识状态 (0/1) |
| `sourceResearchGroup` / `author` / `accountName`| TEXT | 数据来源课题组 / 上传者 / 账户名 |
| `sourcePath` | TEXT | 数据来源路径 (如 "clickhouse") |
| `referenceCitation` | TEXT | 参考文献/引用 |
| `delFlag` | INTEGER | 逻辑删除标记 (Soft Delete Flag) |
| `collectTime` | INTEGER | 采集或入库时间戳 |
| `redundantFields1`~`3` | TEXT | 冗余/扩展备用字段 |
